# create a SKILL.md file available as a plugin

Example marketplace file
```
❯ cat .claude-plugin/marketplace.json
{
  "name": "steadily-cli",
  "metadata": {
    "description": "Marketplace for internal Steadily plugins"
  },
  "owner": {
    "name": "Steadly Internal",
    "email": "dev@steadily.com"
  },
  "plugins": [
    {
      "name": "steadily-cli",
      "description": "steadily-cli usage skills",
      "source": "./plugins",
      "version": "0.1.135",
      "author": {
        "name": "Steadily Engineering"
      },
      "skills": [
        "./steadily-cli"
      ]
    }
  ]
}
```

And the SKILL
```
❯ head -20 plugins/steadily-cli/SKILL.md
---
name: steadily-cli
description: >
  Command-line interface for retrieving Steadily insurance policy information, billing data, and quote history.
  Use when working with Steadily policies to:
    (1) Get detailed policy information including coverages, premiums, and underwriting details
    (2) Look up statement accounts and billing information
    (3) Retrieve invoice details and payment links
    (4) View policy quote history and rate changes.
allowed-tools: Bash(K8S_NAMESPACE=prod steadily-cli policy debug:*), Bash(K8S_NAMESPACE=prod steadily-cli quote history:*), Bash(K8S_NAMESPACE=prod steadily-cli sa sync:*), Bash(K8S_NAMESPACE=prod steadily-cli sa get:*)
---
```
